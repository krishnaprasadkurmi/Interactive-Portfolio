# 🌐 Interactive Portfolio Website

## 📌 Project Overview

This project is an **Interactive Portfolio Website** developed using **HTML5, CSS3, and JavaScript**. The purpose of this project is to demonstrate JavaScript fundamentals by creating an attractive and responsive personal portfolio with interactive features.

The website includes sections such as Home, About, Skills, Projects, Todo List, and Contact. JavaScript is used to make the website dynamic and user-friendly through DOM manipulation, event handling, animations, form validation, and local storage.

---

# 🎯 Objectives

- Build a responsive portfolio website.
- Learn JavaScript fundamentals.
- Implement DOM Manipulation.
- Handle user events using Event Listeners.
- Validate user input using JavaScript.
- Store user preferences using Local Storage.
- Create reusable JavaScript functions.
- Improve user experience with interactive elements.

# 🖱️ Event Listeners Used

| Event | Purpose |
|---------|---------|
| click | Button actions |
| submit | Form Validation |
| scroll | Scroll to Top Button |
| mouseenter | Hover Animation |
| mouseleave | Card Effects |
| load | Load Saved Theme |

---

# ✔️ Form Validation

Validation Rules:

### Name

- Minimum 3 characters

### Email

- Valid Email Format

### Message

- Minimum 10 characters

If validation fails, error messages are displayed.

If validation passes, a success message is displayed.
